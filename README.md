## Общие данные, пошаговая инструкция.

Настоящая инструкция описывает порядок подготовки модели и выполнения расчета инсоляции или затенения в среде Revit с использованием скрипта.

### 1.1.Принцип-расчета.

Метод расчета количества часов инсоляции сформирован на основе **“СанПиН 2.2.1/2.1.1.1076-01 Гигиенические требования к инсоляции и солнцезащите помещений жилых и общественных зданий и территорий.**” Указанный СанПин предполагает использовать **графический метод** \- инсоляционную линейку, для расчета инсоляции в определенной точке.
Приведенный метод использует **геометрический метод** для анализа геометрии и расчета инсоляции посредством скрипта Dynamo, однако все еще требует указания точки для который необходимо провести расчет. В рамках методики принято использовать семейство Revit, геометрический центр которого будет приниматься за расчетную точку.

Принципиально геометрический анализ заключается в определении наличия пересечения луча, исходящего из расчетной точки по направлению координат солнца, определенных относительно координат проекта с учетом с даты и времени, с любой геометрией окружающей расчетную точку.

Принципиально геометрический анализ заключается в определении наличия пересечения луча, исходящего из расчетной точки по направлению координат солнца, определенных относительно координат проекта с учетом с даты и времени, с любой геометрией окружающей расчетную точку.

Вычисления координат солнца основаны на методике NOAA, погрешность результата от калькуляторов NOAA составляет ≈ 1°.

Сайт NOAA \- [NOAA](https://www.noaa.gov/)  
Калькулятор 1 \- [NOAA Solar Position Calculator](https://gml.noaa.gov/grad/solcalc/azel.html)  
Калькулятор 3 \- [Solar Calculator \- NOAA Global Monitoring Laboratory](https://gml.noaa.gov/grad/solcalc/)

Перед запуском скрипта необходимо убедиться, что модель подготовлена корректно.  
Для расчета необходима корректная трехмерная геометрия окружающей застройки и проектируемого объекта.

Результатом работы является графическое отображения диапазона иснолирующих лучей для рассчитываемой точки, а также расчет времени инсоляции исследуемых помещений или объектов в минутах.

Расчет следует проводить в файле, в котором размещены рассчитываемые помещения.
